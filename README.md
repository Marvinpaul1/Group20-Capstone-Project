# Planetary Data Visualizer (Group 20 Capstone Project)

A modern React web application that visualizes fascinating planetary data fetched from Wikipedia Open APIs. This project was built by Group 20 as the final capstone project for the TS Academy program.

## 🚀 Features

- **Live Wikipedia Data**: Integrates with Wikipedia's Open APIs to provide real-time planetary information.
- **Interactive Visualizations**: Clean UI to explore data points about our solar system.
- **Responsive Design**: Built to work on desktops, tablets, and mobile devices.
- **Modern Stack**: Developed using React 18 features and Vite for lightning-fast development.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: JavaScript / TypeScript
- **API**: [Wikipedia Open APIs](<https://en.wikipedia.org/api/rest_v1/page/summary/${encodeURIComponent(page)}`>)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v16.0.0 or higher recommended)
- npm or yarn

## ⚙️ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-repo/Group20-Capstone-Project.git
   cd Group20-Capstone-Project-main
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open http://localhost:5173 in your browser to see the result.

## 🏗️ Available Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Locally previews the production build.
